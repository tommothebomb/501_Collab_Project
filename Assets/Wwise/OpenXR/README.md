# Wwise Unity OpenXR Motion

This package adds support for Wwise Motion haptics on Meta Quest devices via OpenXR.

## Requirements

* Unity 2021.3 or later
* Unity XR Plugin Management with OpenXR plugin >= 1.16.0
* Wwise Unity Integration
* Wwise Motion plugin

## Usage
See [Unity Wwise Motion OpenXR Documentation]() for more information about how to use the package.
